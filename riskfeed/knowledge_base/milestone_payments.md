# Milestone-based payments (why it matters)

In a milestone-based payment model, funds are released only when a milestone is completed and verified.
This reduces deposit misuse and improves project accountability.

Recommended practices:
- Break scope into clear line items (design, demolition, rough-in, finish)
- Define "Definition of Done" for each milestone
- Require evidence: photos, inspection reports, receipts (as appropriate)
- Dispute workflow: pause release until both parties agree or escalation occurs
